# Frenzy INK — Website

Multi-page static website for **Frenzy INK**, a tattoo and piercing shop in Johnson City, TN.

- Pure HTML / CSS / JavaScript — no build step
- Animations powered by [anime.js](https://animejs.com/)
- Fully responsive (mobile-first refinements included)
- Tattoo-flash aesthetic: grain overlay, flash-sheet borders, star/dagger ornaments, hand-drawn script accents

## Pages

- `index.html` — Hero, features, new-owner highlight, CTA
- `about.html` — Shop story and new-ownership announcement (Justin Dotson)
- `services.html` — Tattoo styles, piercings, aftercare
- `artists.html` — Owner spotlight + full crew
- `gallery.html` — Filterable portfolio with lightbox
- `contact.html` — Info, booking form, embedded map

## Run locally

```powershell
python -m http.server 8000
```

Then open http://localhost:8000/.
